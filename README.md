# 🏥 MediAssist AI – Medical RAG Assistant

An AI-powered medical question-answering system using Retrieval-Augmented Generation (RAG), FastAPI, ChromaDB, and Google Gemini.

---

## 🔥 Features

- 🧠 RAG-based medical Q&A system
- 📚 Vector search using ChromaDB
- 🤖 LLM responses via Gemini API
- ⚡ FastAPI backend
- 💬 React chat UI
- 🔐 Secure API key handling via environment variables
- 📄 Source-grounded answers from medical documents

---

## 🏗️ Architecture

User Query → Frontend (React) → FastAPI Backend →  
Vector DB (ChromaDB) → Context Retrieval → Gemini LLM → Response

---

## 🧰 Tech Stack

- Frontend: React + Tailwind (or your setup)
- Backend: FastAPI
- LLM: Google Gemini API
- Vector DB: ChromaDB
- Embeddings: SentenceTransformers (MiniLM)
